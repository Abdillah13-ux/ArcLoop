# ArcLoop Demo Script

Target video duration: 3-4 minutes.

## Demo Flow

1. Open ArcLoop mobile app.
2. Login with Google through Circle User-Controlled Wallets.
3. Show connected wallet address.
4. Create a rotating savings pool.
5. Invite members to join the pool.
6. Start the pool when member count is full.
7. Members contribute USDC for round 1.
8. Release pooled payout to the predefined recipient.
9. Show payout proof in the app.
10. Open Arc explorer link for contract and transaction verification.

## Reviewer Proof Points

- Mobile-first user experience.
- Circle Wallets onboarding.
- Arc Testnet smart contract.
- USDC contribution.
- USDC payout.
- Verifiable transaction hash.
- No random winner, no yield, no gambling mechanics.
